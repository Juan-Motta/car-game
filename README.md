# Guia de uso

Juego de carros por consola creado por [Juan Motta](https://github.com/Juan-Motta) y escrito en TypeScrript.

## Requisitos

* NodeJS

## Ejecucion

Dentro de la carpeta contenedora .../car-game/ ejecutar el comando
```
npm run start
```
